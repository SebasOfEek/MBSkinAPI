MBSkinAPI

Interactive Quick Start Guide - Get Started in 3 Steps

Getting Started

1

Add the CSS Stylesheet

Copy this line into the <head> section of your HTML file:

HTML 
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/SebasOfEek/MBSkinAPI@main/dist/MBSkinStyles.css">
2

Add the JavaScript Bundle

Copy this line before the closing </body> tag:

HTML 
<script src="https://cdn.jsdelivr.net/gh/SebasOfEek/MBSkinAPI@main/dist/MBSkinAPI.bundle.min.js"></script>
3

Initialize the API

Add this JavaScript code to create your skin explorer:

JavaScript 
// Initialize the skin explorer
const skinAPI = new MBSkinAPI({
    onSkinSelect: function(skin) {
        console.log('Selected skin:', skin);
        // Your custom logic here
    }
});

// Open the explorer
skinAPI.show();

Built-in search functionality with filtering by name or author.
