¿Cómo funciona el CDN?

El CDN (Content Delivery Network) es una forma de distribuir los archivos del API servidores rápidos alrededor del mundo. En lugar de descargar los archivos desde tu proyecto, los usuarios las descargan desde el CDN más cercano.

Ejemplo de uso con CDN:
<script src="https://cdn.jsdelivr.net/gh/tu-usuario/MBSkinAPI@main/dist/MBSkinAPI.bundle.min.js"></script>

URLs de GitHub para CDN:
• jsDelivr: https://cdn.jsdelivr.net/gh/usuario/repo@rama/src/archivo.js
• Statically: https://cdn.statically.io/gh/usuario/repo/rama/src/archivo.js
• GitHack: https://rawcdn.githack.com/usuario/repo/rama/src/archivo.js
